Bulk CSV Upload

Upload a CSV file with the following required columns:

origin_code, destination_code, container_type, pricing_tier, rate_value, valid_from, valid_to

Optional column: transit_time

Example row: LKCMB, SGSIN, 20_dry_standard, public, 850.00, 2026-01-01, 2026-06-30, 5-7 days
Validation rules (entire upload fails if any row is invalid):
  • All required columns must be present in the header
  • Origin & destination port codes must exist in the system
  • Container type must match an active container type
  • Pricing tier must match an active pricing tier
  • Rate value must be a positive number
  • Dates must be valid (YYYY-MM-DD) and valid_from ≤ valid_to
  • No duplicate rows (same origin + destination + container + tier + date range)
